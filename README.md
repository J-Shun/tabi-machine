# Tabi Machine

旅遊行程規劃 Line LIFF 應用 (暫時僅支援手機版畫面)

## 功能特色

### 行程管理

- **建立行程**：輕鬆建立旅遊行程
- **編輯行程**：修改行程名稱、日期和封面照片
- **刪除行程**：移除不需要的行程

### 詳細行程規劃

- **日期自動排列**：根據旅行日期自動生成每日行程
- **項目分類**：支援景點、住宿、美食、購物、交通、其他等類型
- **拖拽排序**：直觀的拖拽功能，輕鬆調整行程順序
- **跨日移動**：可將行程項目移動到不同日期
- **時間軸視圖**：清晰的時間軸展示每日行程

### 實用功能

- **地圖整合**：一鍵開啟 Google Maps 查看地點
- **今日定位**：自動滾動到當天的行程
- **本地儲存**：資料保存在本地，快速存取
